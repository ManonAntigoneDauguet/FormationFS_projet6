# MDD (Monde de Dév)

Welcome to this "Minimum Viable Product" for the MDD ("Monde de Dév") project.  
This is a fullstack project with Java Spring Boot and Angular technology.

## Dependencies

This project is using java 17, MySQL and Angular.
